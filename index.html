3 Modal Examples CSS-Only

These are common dashboard patterns: Delete confirm, Edit form, and Login. They are built with only HTML and CSS, no JavaScript using :target.

Open Delete Confirm Open Edit Form Open Login

Recruiter note: This is a demo of how to create modals using only HTML and CSS. The modals are triggered using the :target pseudo-class, which allows us to show and hide the modal content without any JavaScript. Each button opens a different modal, and you can close the modal by clicking outside of it or using the close button inside the modal. A production modal would add JS for Escape and focus management, but this is a learning demo of how to do it with just HTML and CSS.
